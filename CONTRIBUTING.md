# Contributing

Thanks for your interest in contributing to the Groundwater Simulator.

Please fork the repository and open a pull request with a clear description of the change.

Guidelines:
- Write tests for new functionality and run the test suite.
- Follow the existing code style. Use `ruff` and `black` for formatting and linting.
- Add documentation to `docs/` for new features or APIs.
- Keep PRs small and focused.

Maintainers will review and provide feedback.
