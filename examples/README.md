# Examples

This directory contains standalone scripts and tutorials for using the Groundwater Simulator.

- **local_test_app.py**: Educational example with basic homogeneous domain and analytical comparison.
- **local_test_app2.py**: Validation example with point sources and Green's function comparison.

Both scripts use Matplotlib for plotting and do not require Streamlit.

To run:
```bash
python examples/local_test_app.py
```
